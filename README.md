# 🏥 Healthcare Analytics Dashboard – Power BI

> An interactive Power BI dashboard that transforms raw healthcare data into actionable insights for hospital performance monitoring, financial analysis, and operational efficiency.

---

## 📌 Project Overview

The healthcare sector operates in a data-rich but insight-poor environment. Large amounts of patient, revenue, and operational data are generated daily, yet decision-making often lacks real-time analytical support.

This project develops a **Power BI dashboard** designed to bridge that gap — providing hospital management and stakeholders with a centralized, interactive analytics system for evaluating hospital efficiency and financial performance.

---

## 📁 Dataset

- **Source:** [Kaggle – Healthcare Dataset by Prasad22](https://www.kaggle.com/datasets/prasad22/healthcare-dataset)
- **Records:** 40,235 patient records from multiple hospitals and healthcare providers
- **Attributes:** 16 key fields including:

| Category | Fields |
|---|---|
| Patient Info | Name, Age, Gender, Blood Type |
| Medical | Medical Condition, Medication, Test Results |
| Hospital | Doctor, Hospital, Admission Type |
| Financial | Insurance Provider, Billing Amount |
| Time-based | Date of Admission, Discharge Date, Total Days |

---

## ❓ Problem Statement

- No centralized view of hospital performance
- Difficult to evaluate financial health and operational efficiency
- Limited visibility for stakeholders and potential investors
- Need for structured analytics to evaluate healthcare performance in a region

---

## 🎯 Objectives

- Monitor admissions and patient stay duration
- Study insurance provider contribution
- Compare disease distribution by gender
- Evaluate doctor workload across admission types
- Enable interactive, real-time data analysis using slicers

---

## 🔧 Tools & Technologies

- **Power BI Desktop** – Dashboard development & visualization
- **Microsoft Excel** – Data preparation and cleaning
- **DAX (Data Analysis Expressions)** – KPI calculations
- **Power Query** – Data transformation

---

## 🧹 Data Preparation

- Cleaned missing and inconsistent values
- Verified and corrected data types
- Created calculated KPI measures using DAX
- Designed structured, interactive dashboard layout
- Connected slicers for dynamic filtering

---

## 📊 Dashboard Features

### 🔢 KPI Cards
| KPI | Description |
|---|---|
| Total Revenue | Overall financial performance |
| Average Bill | Revenue per patient |
| Total Admissions | Hospital demand indicator |
| Average Days Spent | Operational efficiency metric |

### 📈 Visualizations

| Visual | Purpose |
|---|---|
| Pie Chart | Insurance provider distribution & financial dependency |
| Donut Chart | Admission type breakdown (Emergency / Urgent / Elective) |
| Clustered Bar Chart | Disease distribution comparison across gender |
| Area Chart | Revenue vs doctor availability trends |
| Tree Map | Medication usage and treatment category proportion |
| Clustered Column Chart | Doctor workload across admission types |

### 🎛️ Interactivity
- Dynamic slicers for filtering by revenue and average stay duration
- Real-time updates across all visuals on slicer selection
- No dataset modification needed — all filtering is visual and interactive

---

## 💡 Key Insights

- 📈 Revenue is directly influenced by admission volume
- 🏥 Certain insurance providers dominate patient coverage
- 🚨 Emergency admissions form a significant share of total admissions
- 👥 Disease patterns vary noticeably by gender
- 👨‍⚕️ Doctor workload distribution differs across admission types
- 💊 Medication usage is concentrated in specific treatment categories
